Highlighter

This plugin allows you to point out some text inside your page. It is fully done with javascript only, so you don't have to include other libraries.

Declaration

var highlighter = new Highlighter();

Usage

Define backgrounds: addStyle(className, backgroundColor).

highlighter.addStyle("red", "rgba(255, 0, 0, .5)").addStyle("green", "rgba(0, 128, 0, .5)").addClass("white", "#FFFFFF");

Now use them:

byID(id, start{optional}, end{optional})

byClass(className, start{optional}, end{optional})

highlighter.style("red").byID("p1", 0, 19); //highlight from 0 to 19
highlighter.style("red").byID("p1", 12); //highlight from 12 to end of the text inside p1
highlighter.style("red").byID("p1"); //highlight entire p1

Examples

highlighter.style("red").byID("p1", 0);

This text has a red background.

highlighter.style("red").byID("p2", 25);

This text, starting from here has a red background.

highlighter.style("red").byID("p3", 5, 18);

From here I am red but now I become normal again.

highlighter.style("red").byID("p4", 5, 18).style("green").byID("p4", 5, 18);

From here I have green and red. But here only green.
